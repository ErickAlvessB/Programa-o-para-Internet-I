# 💻 Programação para Internet I

Repositório destinado às atividades, projetos e estudos da disciplina **Programação para Internet I**, do curso de **Análise e Desenvolvimento de Sistemas (ADS)** do Instituto Federal do Piauí (IFPI) – Campus Angical.

## 📍 Informações da Disciplina

- **Instituição:** Instituto Federal do Piauí (IFPI)  
- **Campus:** Angical  
- **Curso:** Análise e Desenvolvimento de Sistemas (ADS)  
- **Disciplina:** Programação para Internet I  
- **Professor:** Warlles Machado  

## 🎯 Objetivo

Entender e aplicar tecnologias de desenvolvimento para a Internet no desenvolvimento de softwares Dinâmicos para a WEB.

## 📚 Conteúdos Abordados

- Estrutura básica de páginas HTML  
- Semântica HTML  

## 📁 Estrutura do Repositório

- atividades/ → Exercícios propostos  
- projetos/ → Projetos desenvolvidos  
- README.md → Documentação do repositório  

## 🚀 Tecnologias Utilizadas

- HTML5
- Git/GitHub: Controle de versionamento e hospedagem de código.
